# Debugging

Follow a disciplined debugging workflow.

## How It Works

1. Reproduce or characterize the failure.
2. Isolate the smallest failing path.
3. Identify the root cause before changing code.
4. Apply the smallest safe fix.
5. Verify with tests or a focused smoke check.
